P126 AOI Circuit

DE - Ms. Chou - Period 6

Project Dates: 10/20/2025 - 10/31/2025

Full Project Documentation ↗

Our project aimed to develop a smart traffic light control system using AOI logic gates to manage flow at a T‑intersection. We built a working prototype on breadboards using switches as sensors, LEDs as signals, and ICs to handle all the logic for detecting cars and changing lights. The system followed the required setup: 8 inputs for lane activity and right‑of‑way rules, 4 outputs for the light signals, all completed within 2 weeks while avoiding unsafe situations like conflicting greens.

I wasn’t the project manager, but I handled my entire output section from start to finish. I created the full truth table for my signal, wrote the Boolean expression, and simplified it using basic Boolean laws so it fit the ICs we had. I wired my breadboard section, tested each gate, and helped debug the full system once all outputs were connected. I also worked with the team to make sure our logic matched the final Multisim simulation.

Working with simple ICs and breadboards taught me how to make limited hardware behave consistently. I had to keep the logic clean, avoid extra gates, and make sure my output didn’t interfere with the others. Even though this setup wasn’t industrial, it still felt similar to real traffic systems where you work within constraints to improve safety and reduce congestion.

At first, some outputs were wrong because a few gates were wired incorrectly. I helped fix this by tracing each connection, checking gate outputs with LEDs, and rebuilding messy wiring using color‑coding so we could follow the signals more easily. After a few rounds of testing, the LEDs finally behaved correctly.

Our switches also gave inconsistent readings at first because one IC wasn’t fully plugged in. I tested each input on its own, found the issue, and reorganized the wiring until everything worked reliably. Throughout the project, I focused on solving logic mistakes and hardware problems, which helped our team get the system working as intended.
